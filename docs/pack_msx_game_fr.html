ℹ️ Foire aux questions
WebMSX est un émulateur MSX complet fonctionnant dans le navigateur, créé par Paulo Augusto Peccin. Il émule les ordinateurs MSX1, MSX2, MSX2+ et MSX turbo R avec une grande fidélité, incluant le son, les extensions matérielles et les différents types de médias (cartouches, disquettes, cassettes). Le BIOS utilisé est C-BIOS, un BIOS open-source pour MSX.
Le packer supporte les formats suivants :
• .rom, .mx1 — Cartouches ROM MSX1/MSX2
• .mx2 — Cartouches ROM MSX2
• .dsk — Images disquettes MSX
• .cas — Images cassettes MSX
• .zip — Archives contenant un des formats ci-dessus (détection automatique)
